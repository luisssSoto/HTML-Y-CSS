Modelado de caja 2da parte

A pesar de que las etiquetas div tienen el mismo valor para el atributo width, existe una diferencia en el valor de la propiedad box-sizing

box-sizing: content-box;
box-sizing: border-box;

Nota: Como se puede observar el primer div tiene mas width que su etiqueta hermana y esto es porque el valor de la propiedad box-sizing tiene una anchura de 200px solo del content pero la segunda etiqueta div tiene un width de 200px tanto para content como para el border
